My FLow App 
A clean vanilla JS mobile-first task manager app built with no framework: for my personal use to learn more about UI/UX

This project focuses on: 

- modular JS architecture
- dynamic UI rendering
- project-based task filtering
- persistent task state using localStorage
- mobile-friendly interaction design


Features

- dynamic project switching
- task filtering buy active project
- interactive task status updates
- persistent data with localStorage
- fully modular JS structure
- mobile-friendly UX
- no async complexity


Tech Stack
- HTML5
- CSS3
- Vanilla JS
- localStorage
- Vercel deployment
